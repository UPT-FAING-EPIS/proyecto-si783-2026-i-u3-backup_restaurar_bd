# Prerrequisitos del Sistema
Node.js >= 18.0.0
Rust >= 1.70.0 (stable)
Cargo (incluido con Rust)

# - TypeScript
# - Vite
# - Tailwind CSS v4
# - Tauri CLI

# - rusqlite (SQLite)
# - serde
# - ring (para encriptación AES-256-GCM)
